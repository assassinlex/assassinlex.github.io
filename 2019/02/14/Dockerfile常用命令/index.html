title: Dockerfile常用命令
date: 2017-12-26 16:05:40
tags: Docker

我们创建一个 Docker 容器，希望其可以自动化执行，不用每次在命令行里面输入一大堆参数，那样很麻烦。Dockerfile提供了一系列指令，可以让我们像执行脚本程序一样，预先定义好容器镜像。接下来简单介绍一下，常用的一些 Dockerfile 命令。

FROM

自定义镜像一般都会有一个基础镜像，FROM 指定一个基础镜像。

FROM alpine

RUN

镜像是一层层构建的，一个镜像可以有多个 Layer。RUN 在镜像构建前运行命令

RUN apt-get install -y vim

CMD

与 RUN 不同的是，CMD 是在容器启动后执行命令。

CMD “ping”

ADD

将宿主机的文件系统复制到容器的文件系统。

ADD host/source docker/destination

COPY

将宿主机的文件系统复制到容器的文件系统。[ COPY 与 ADD 相差不大，前者支持远程资源(URL) ]

ADD host/source docker/destination

ENV

指定当前容器的环境变量

ENV PHP_VERSION 5.3

VOLUME

把你的磁盘上的数据挂载到容器的挂载点上，这个修改了宿主机文件直接反映到容器里面。

VOLUME host/dir docker/dir

EXPOSE

将容器内暴露给外部的端口

EXPOSE 22

以上介绍的就是用的频率比较频繁的 Dockerfile 指令了，更多指令请查阅官网。
